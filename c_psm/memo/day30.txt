함수
	이름 뒤에 소괄호()가 있으면 함수이다.
	단, 키워드 뒤에 소괄호가 있으면 함수가 아니다.
	ㅇ)if(), switch(), for(), while(),...

함수의 요소
	f	(x)	=	2x+1
	함수명	매개		리턴값
		변수

함수의 선언
	(1)리턴타입 (2)함수명(자료형 (3)매개변수1,...)
	{
		(4)싱행할 문장;
		(5)return 리턴값;
	}

(1) 리턴 값의 타입을 작성한다. 만약 리턴할 값이 없다면 비워놓지 않고 void라는 타입을 작성해준다.
(2) 함수 이름은 동사로 작성한다(연필(매개변수)을 쓴다(함수)).
(3) 생략이 가능하지만 외부에서 값을 전달받기 위해서는 반드시 자료형과 순서에 맞게 선언해주어야 한다.
(4) 생략 가능, main함수에서 작성해왔던 소스코드들.
(5) 생략 가능

함수 선언과 사용 구분
	{}가 있으면 선언, {}가 없으면 사용

함수 선언 시 주의사항
	함수 안에 다른 함수를 선언할 수 없으며, 함수 사용은 반드시
	함수 안에서 사용해야 한다.
	함수를 사용할 때 선언부가 소스코드 밑에 있으면 인식이 안되기 때문에,
	반드시 함수 선언은 사용하는 부분 위에 선언되어야 한다.
